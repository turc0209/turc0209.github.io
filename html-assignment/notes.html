HTML Tags

a (anchor tag)
Accompanied by href, this tag creates a hyperlink to anything a URL can address.
p (paragraph tag)
This tag creates a block of text allocated for paragraphs.
nav (navigation tag)
This tag allocates space for navigation items on the page.
ul (unordered list tag)
This tag, accompanied by li items creates a format for an unordered list.
img (image tag)
This tag, accompanied by src and alt display an image on the page and provide alt text for accessibility.
